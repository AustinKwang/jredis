jredis by alphazero

Java Client and Connectors for Redis

Dependencies

JUnit 3 is used during the build/test process. It is not required for runtime environment. System is Java 5.x+

Install

Maven 2 build management * "mvn -version" will let you if you have maven and at what revision. * to compile, test, package (generate all module jars), and install in your local repository, simply type "mvn install" from the root or core package. ** Note ** that the build will run unit tests against a localhost redis server, ** flushing db #10 and #13 repeatedly ** using password "jredis". If you do not have password protection for your redis, that's fine. if you do, the build will fail. (You can modify the poms as you see fit for your needs, but the project depends on these tests and the master poms will not be modified.)
